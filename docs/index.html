⚗ IOC Pattern Scanner

Paste content from a suspicious file. The scanner runs the same patterns as the triage script.

Detection Patterns
API Keys: sk-* · sk-ant-* · AIza* · pcsk_* · hf_*
Endpoints: openai · anthropic · azure · google
Injection: ignore previous · ChatML · jailbreak
Exfil: private keys · SSN · CC · keywords
ⓘMinimum logging controls every organization should enable.
