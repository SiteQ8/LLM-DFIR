⚙ Triage Command Builder

Configure and generate a ready-to-run triage command for the target host. Copy the output and run it in the analyst console.

⚠ Run on the target host with the privileges of the user being investigated. The script reads files from the home directory and may capture sensitive content.

Useful one-liners for common analyst tasks during an active LLM incident.

$ grep -h '"method":"tools/call"' ~/Library/Logs/Claude/mcp-server-*.log \ | jq -r '.params.name' | sort | uniq -c | sort -rn
